# 📌 Laborator: Salvare date în fișiere și testare cu PyUnit

## 📖 Enunț

Modificați aplicația realizată la laboratoarele **7-9** astfel încât datele aplicației să poată fi salvate și încărcate din fișiere text.

Aplicația trebuie să poată funcționa în două moduri:

- cu date salvate **în memorie**
- cu date salvate **în fișiere text**

Alegerea modului de lucru se face din **application coordinator**.

---

## ⚙️ Cerințe generale

- Adăugați posibilitatea de salvare a datelor în fișiere text
- Creați clase `Repository` care salvează în fișier
- Modificați `application coordinator` astfel încât aplicația să poată folosi:
  - repository în memorie
  - repository în fișiere
- Folosiți framework-ul **PyUnit** pentru testare
- Transformați funcțiile de test existente în clase `TestCase`
- Realizați **white-box testing** pentru fiecare metodă
- Pentru o metodă la alegere realizați și **black-box testing**
