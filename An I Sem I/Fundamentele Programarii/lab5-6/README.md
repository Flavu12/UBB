# 📌 Proiect: Gestionare numere complexe

## 📖 Descriere

Acest proiect implementează operații asupra unei colecții de numere complexe de forma **a + bi**.

Inițial, aplicația a fost realizată folosind o **listă** pentru stocarea numerelor complexe.  
Ulterior, cerința a fost să se transforme structura de date într-un **dicționar**.
